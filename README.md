PrintJob
========

/!\ Only works with Android at the moment.
PrintJob for OpenFL, lets you print with your device

##Use

add PrintJob in haxelib.
Then :

  ```haxe
  PrintJob.printBitmapFile("valid image file", 'id of the printjob');
  ```
